# Sprint Roadmap: UACOS v4 To Beta

## Goals

1. Keep documentation navigable and reviewable.
2. Preserve CLI backward compatibility.
3. Harden MCP hotfix behavior.
4. Validate cache similarity and TTL compatibility.
5. Require a passing release gate before any release artifact.

## Non-Goals

No Redis, PostgreSQL, RabbitMQ, Celery, Pinecone, GitHub PR bot, or multi-tenant work in this sprint.
